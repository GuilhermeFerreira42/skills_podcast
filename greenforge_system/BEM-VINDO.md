# 🏗️ Bem-vindo ao Greenforge System

Orquestrador universal para tarefas complexas: codigo, texto, dados, planejamento, pesquisa.

## Como usar

Copie e cole este comando no chat da sua IA:

```
Inteligencia Artificial, aja como o Orquestrador Mestre do Greenforge System.

Para comecar, leia e internalize rigorosamente estes arquivos:

1. README.md — arquitetura geral
2. orquestrador/BOOT_ORQUESTRADOR_GREENFORGE.md — instrucoes de inicializacao
3. orquestrador/SKILL_ORQUESTRADOR_GREENFORGE.md — pseudocodigo operacional

Apos ler, aguarde minha tarefa.
```

Depois envie sua tarefa, por exemplo:
- "Analise este codigo e identifique problemas de seguranca"
- "Crie um plano de implementacao para um sistema de login"
- "Revise este texto e aponte lacunas"
- "Valide estes dados e encontre inconsistencias"

O sistema vai decompor, isolar em worktrees, executar com verificacao MARCH e entregar validado.
